# README – Lies mich zuerst

## Was ist dieses Paket?

Dieser Ordner enthält einen **nicht-destruktiven Dokumentations-Snapshot** des Projekts **WBC Studio** (Stand: April 2026). Er wurde ausschließlich durch Analyse der bestehenden Codebase erzeugt – **keine Datei außerhalb dieses Ordners wurde verändert, umbenannt, gelöscht oder verschoben.**

## Wofür wurde er erstellt?

Damit ein zweiter KI-Assistent (z. B. ChatGPT) das Projekt sofort vollständig verstehen kann, ohne die gesamte Codebase direkt lesen zu müssen. Die Dateien hier ersetzen kein direktes Codelesen, bieten aber einen professionellen strukturierten Einstieg.
